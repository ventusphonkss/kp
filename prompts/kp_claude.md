# ПРОМПТ ДЛЯ СОЗДАНИЯ ШАБЛОННОГО КОММЕРЧЕСКОГО ПРЕДЛОЖЕНИЯ

## Анализ изученного КП

Изучил ваше КП от No Logo Studio. Это качественный пример структурированного коммерческого предложения для digital-агентства с детальной разбивкой работ, стоимости и информацией о компании.

---

## ПРОФЕССИОНАЛЬНЫЙ ПРОМПТ ДЛЯ ИИ

### Основной запрос:

```
Создай профессиональный шаблон коммерческого предложения для дизайн-агентства 
в формате, который легко адаптировать под разные проекты.
```

---

## СТРУКТУРА ШАБЛОНА:

### 1. ШАПКА ДОКУМЕНТА
- Логотип и название агентства
- Контактные данные (телефон, email, сайт)
- Слоган компании
- Дата создания КП

### 2. ЗАГОЛОВОК
**"Коммерческое предложение на разработку [ТИП УСЛУГИ]"**

Для компании: **[НАЗВАНИЕ КОМПАНИИ КЛИЕНТА]**

### 3. ЗАДАЧА
- Краткое описание запроса клиента
- Цели проекта
- Целевая аудитория (если применимо)
- Особые требования

### 4. ОПИСАНИЕ УСЛУГИ
- Что входит в работу
- Этапы выполнения
- Количество концептов/вариантов
- Количество правок
- Форматы файлов на выходе

### 5. СОСТАВ РАБОТ (ДЕТАЛИЗАЦИЯ)

**Таблица с разбивкой:**
- Этап работы
- Описание
- Специалист
- Количество часов
- Стоимость за час
- Итого по этапу

### 6. СРОКИ РЕАЛИЗАЦИИ
- Общий срок проекта
- Сроки по каждому этапу
- Условия продления сроков

### 7. СТОИМОСТЬ
- Общая стоимость проекта
- Условия оплаты (предоплата/постоплата)
- Что НЕ входит в стоимость (если есть)

### 8. КОМПЛЕКТАЦИЯ ПРОЕКТА

**Что получает клиент:**
- Исходные файлы
- Форматы
- Гайдлайны/брендбук (если применимо)
- Техническая документация

### 9. ГАРАНТИИ
- Срок гарантийной поддержки
- Что включает гарантия
- Условия внесения правок

### 10. О КОМПАНИИ
- Краткая информация об агентстве
- Опыт работы
- Ключевые клиенты
- Награды/сертификаты
- Портфолио (2-3 кейса)

### 11. КОНТАКТЫ
- Ответственное лицо
- Телефон, email
- Призыв к действию

---

## ТРЕБОВАНИЯ К ОФОРМЛЕНИЮ:

✓ Минималистичный дизайн  
✓ Четкая структура  
✓ Легко читаемые шрифты  
✓ Корпоративные цвета  
✓ Профессиональный тон текста  
✓ Объем: 3-5 страниц максимум

## ПЕРЕМЕННЫЕ ДЛЯ ЗАПОЛНЕНИЯ

**Выдели в шаблоне следующие поля:**

- `[НАЗВАНИЕ_КОМПАНИИ_КЛИЕНТА]`
- `[ТИП_УСЛУГИ]` (логотип, фирменный стиль, упаковка и т.д.)
- `[ОПИСАНИЕ_ЗАДАЧИ]`
- `[КОЛИЧЕСТВО_КОНЦЕПТОВ]`
- `[КОЛИЧЕСТВО_ПРАВОК]`
- `[СРОК_ВЫПОЛНЕНИЯ]`
- `[ОБЩАЯ_СТОИМОСТЬ]`
- `[УСЛОВИЯ_ОПЛАТЫ]`
- `[ДАТА_КП]`
- `[ОТВЕТСТВЕННОЕ_ЛИЦО]`
- `[КОНТАКТЫ]`

## ИНСТРУКЦИЯ ПО ИСПОЛЬЗОВАНИЮ:

### Шаг 1: Создание базового шаблона
Скопируйте промпт выше и вставьте в ChatGPT/Claude для генерации шаблона

### Шаг 2: Адаптация под проект
Замените переменные в `[ ]` на конкретные данные:
- Название компании клиента
- Тип услуги
- Стоимость и сроки

### Шаг 3: Финальная проверка
- Проверьте все цифры
- Убедитесь в корректности контактов
- Проверьте орфографию

---

## ВАРИАНТЫ СПЕЦИАЛИЗИРОВАННЫХ ШАБЛОНОВ:

Можете создать отдельные шаблоны для каждой услуги:

### 1. Разработка логотипа
- Количество концептов: 2-3
- Правки: до 3 раундов
- Форматы: AI, EPS, PNG, SVG

### 2. Фирменный стиль
- Логотип + элементы стиля
- Брендбук базовый
- Шаблоны документов

### 3. Дизайн упаковки
- Концепты дизайна
- Технические макеты
- Подготовка к печати

### 4. Дизайн сайта
- Прототип
- UI/UX дизайн
- Адаптивная версия

### 5. SMM-дизайн
- Шаблоны постов
- Обложки
- Сторис

## ДОПОЛНИТЕЛЬНЫЕ РЕКОМЕНДАЦИИ:

1. **Сохраните базовый шаблон** как отдельный файл
2. **Создайте библиотеку блоков** для разных услуг
3. **Ведите базу цен** для быстрого расчета
4. **Обновляйте портфолио** в разделе "О компании"
5. **Автоматизируйте** повторяющиеся элементы
